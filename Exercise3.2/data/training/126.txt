**** Overview ****
Prema is a runtime support system for parallel compilers and problem solving
environments that target scientific computing applications. Prema is build on
top of PORTS_(Cornell_implementation) and is designed to run on a variety of
MPP and SMP computers. PREMA suppots:
    * global address space memory model
    * data and task parallel programming models
    * multi-threaded_style_of_execution
    * automatic_work-sharing_mechanism_(dynamic_load_balancing)
