***** General Course Information *****
    * Course Overview
          o Introduction
          o Linear Inequalities and Theorem of the Alternative
          o Convex Sets in $R sup n$
          o Convex and Concave Functions
          o Saddlepoint Optimality Criteria without Differentiability
          o Differentiable Convex and Concave Functions
          o First Order Optimality Criteria with Differentiability
          o Second Order Optimality Criteria with Differentiability
          o Duality in Nonlinear Programming
          o Generalizations of Convex Functions
          o Other Optimality Conditions: Exact Penalty, Augmented Lagrangians,
            Gradient Projection, etc.
